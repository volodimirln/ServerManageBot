Список команд:
/start - запуск бота
/wakeupserver - запуск сервера
/restart - перезагрузка сервера
/shutdown - отключение сервера
/block с параметром - блокировка учетки
/unblock с параметром - разблокировка учетки
/addit с параметром - создание учетки
/addeng с параметром - создание учетки
/delete с параметром - удаление учетки
/resetpass с параметром - сброс пароля на p0ssw0rd
/scan - сканирование сети
/wakeupalldevices - запуск всех компьютеров
/wakeupmac  с параметром - запуск компьютера по адресу
/key с параметром - регистрация
